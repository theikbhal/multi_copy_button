Install and Load the Extension

Since this extension is open source, you can load it directly from your computer in developer mode in a few quick steps:

1

Clone the repository

Get the source code onto your local machine:

git clone https://github.com/theikbhal/multi_copy_button.git
2

Open Extension Settings

Launch Google Chrome, navigate to chrome://extensions/, and toggle "Developer mode" in the top-right corner.

3

Load Unpacked

Click the "Load unpacked" button in the top-left, and select the extension/ directory from the folder you cloned.
